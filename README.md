# SparkReader
## Offline AI. Limitless Learning. In Your Hand.

[SparkReader](https://sparkreader.app/) is an AI-powered ebook reader with contextual explanations, curated library, and image-to-ebook creation, all offline. It was inspired by [Google - The Gemma 3n Impact Challenge](https://www.kaggle.com/competitions/google-gemma-3n-hackathon). See the offcial submission [here](https://www.kaggle.com/competitions/google-gemma-3n-hackathon/writeups/sparkreader) and the demo video [here](https://www.youtube.com/watch?v=HFeY2Dxkqp0).
